# 標準出力について

## 1行出力

```python
# 文字列の出力
print("Hello world")

# 整数の出力
print(100)

# 小数の出力
print(0.5)
```

## 複数行出力


```python
# 文字列の複数行出力
print("A", "B", "C", sep="\n")

# 整数の複数行出力
print(8, 1, 3, sep="\n")

# 小数の複数行出力
print(1.2, 5.0, 7.3, sep="\n")
```

## 四則演算

```python
# 加算
A = 1 + 2 # 3

# 減算
A = 10 - 3 # 7

# 乗算
A = 5 * 2 # 10

# 除算
A = 10 / 2 # 5

# 除算のあまり
A = 5 % 2 # 1

# 累乗
A = 2 ** 4 # 16


```
